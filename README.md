# NovaNest Market SPA

A complete data-driven Vue 3 single-page application using DummyJSON.

## Implemented Features

- Strict TypeScript interfaces (no `any`)
- Product catalog fetch from `/products`
- Search, category filter, and sorting
- Dynamic product details route (`/product/:id`)
- Authentication simulation (`/auth/login`)
- Persistent auth state in `localStorage`
- Persistent cart with Pinia
- Dark mode toggle
- Fully responsive Tailwind UI

## Stack

- Vue 3 (Composition API)
- TypeScript
- Vite
- Tailwind CSS
- Pinia
- Vue Router

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
